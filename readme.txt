== Description ==

Testimonials plugin allows you to display random or selected testimonials, or text with images.
Features
1 Admin interface to add, edit and manage testimonials
2 Capable of handling multiple widgets per page or post
3 Fields for source, testimonial, image, title, location, email, company and URL details
4 Filters to manipulate testimonials output

Shortcodes

[lumia_testimonial_list] - For listings on page or posts
[lumia_testimonial_widget] - For rotating testimonials

Theme Functions

lumia_testimonial_list() - For listings on page or posts
lumia_testimonial_widget() - For rotating testimonials

== Installation ==

1.     Via WordPress Admin > Plugins > Add New, Upload the lumia_testimonial.zip file
2.     Alternately, via FTP, upload lumia_testimonial directory to the /wp-content/plugins/ directory
3.     Activate the 'Lumia Testimonial' plugin after uploading or through WordPress Admin > Plugins

Usage

1. Add and manage the quotes through the 'Testimonials' menu in the WordPress admin area
2. To display testimonials in the sidebar, go to 'Widgets' menu and drag the 'Lumia Testimonials' into the desired widget area
3. Configure the 'Lumia Testimonials' to select quotes and display as needed
4. Use the [lumia_testimonial_list] or [lumia_testimonial_widget] shortcodes to display testimonials on a page or in a post
5. Read FAQ 1 for lumia_testimonial_list() and lumia_testimonial_widget() theme functions usage
